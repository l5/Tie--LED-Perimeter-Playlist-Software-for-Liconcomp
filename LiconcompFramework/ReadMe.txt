Liconcomp .Net Framework Library
================================

Diese .Net Bibliothek kapselt den Zugriff auf die Liconcomp COM Schnittstelle.
Sie vereinfacht das Event Handling und die Erstellung neuer Liconcomp Objekte.

Erstellen Sie eine Instanz von "Player" um Liconcomp zu starten und zu benutzen:

Player P = new Line5.Liconcomp.Player();

Für mehr Informationen lesen Sie bitte die Hilfetexte der Player Klasse.

Hinweis: Das Framework kapselt noch nicht die vollständige Funktionalität von
         Liconcomp!
